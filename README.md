# LCA

LCA 是一个面向 Windows 桌面自动化场景的工作流编排工具。项目提供可视化任务编排、图像识别、OCR、鼠标键盘输入、窗口控制、多窗口执行、插件适配和运行时管理等能力，适合构建需要跨窗口、跨任务步骤协作的本地自动化流程。

> 当前目录是面向开源发布整理后的源码版本。私有授权地址、更新地址、市场服务地址、交流群链接、本地凭据、证书、日志、模型、图片资源和用户工作流数据已从此目录中移除或替换为占位值。

## 功能概览

- 可视化工作流：创建、编辑和执行由多个任务节点组成的自动化流程。
- 图像与 OCR：支持截图、图像匹配、区域识别和 OCR 子进程处理。
- 输入自动化：支持鼠标、键盘、滚轮、录制回放和前台窗口输入。
- 窗口管理：支持窗口选择、绑定、激活、多窗口任务分配和执行控制。
- 多进程运行：OCR、图像匹配、工作流等能力可通过独立子进程隔离运行。
- 插件适配：包含插件接口、插件管理和 OLA Python 适配源码。
- 市场模块：保留市场包模型、导入导出、安装和发布相关源码。
- 主题系统：包含浅色、深色主题和主题管理逻辑。

## 目录结构

```text
.
├── app_core/          # 应用核心配置、授权运行时、日志和插件桥接
├── config/            # 默认配置文件，不包含私有凭据
├── market/            # 市场包管理、导入导出和服务地址配置
├── OLA/               # OLA Python 适配源码，不包含 DLL/CHM/DB
├── plugins/           # 插件接口、管理器和适配器
├── resources/         # 轻量文本资源
├── services/          # OCR、截图、AI、MCP、地图导航等服务模块
├── task_workflow/     # 工作流执行、变量、进程代理和上下文
├── tasks/             # 自动化任务实现
├── themes/            # QSS 样式、主题管理器和主题资源
├── ui/                # PySide6 用户界面
├── utils/             # 输入、截图、窗口、路径、运行时等通用工具
├── main.py            # 应用入口
└── requirements.txt   # Python 依赖清单
```

## 环境要求

- Windows 10 或更高版本。
- Python 3.10，建议使用虚拟环境。
- 依赖安装工具：`pip`。
- 可选：部分截图、输入、OCR、插件和地图导航能力需要额外的系统组件、模型文件或第三方二进制文件。

## 安装与运行

```powershell
python -m venv venv
.\venv\Scripts\python -m pip install --upgrade pip
.\venv\Scripts\pip install -r requirements.txt
.\venv\Scripts\python main.py
```

如果你的系统中 `python` 命令不可用，请替换为本机 Python 3.10 的完整路径。

## 配置

默认配置位于：

- `config/default_config.json`
- `config/universal_system_config.json`

开源版不会内置真实的私有服务地址。以下地址已使用 `https://example.invalid` 作为占位值：

- 授权服务地址。
- 配置服务地址。
- 更新服务地址。
- 市场服务地址。

需要接入自有服务时，请通过环境变量或本地私有配置注入：

- `AUTH_SERVER_URL`
- `CONFIG_SERVER_URL`
- `MARKET_UPDATE_SERVER_BASE`
- `MARKET_VERIFY_SSL`

不要将真实密钥、令牌、证书、账号密码或内部服务地址提交到公开仓库。
