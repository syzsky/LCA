# 共享平台模块骨架

当前目录先提供共享平台系统的基础骨架，不接入现有 UI 和执行链路。

现阶段包含：

- `models.py`：共享平台包清单、运行要求、目标窗口要求、预检结果模型
- `storage.py`：本地共享平台包目录规划
- `package_archive.py`：共享平台包归档读取与基础结构校验
- `environment.py`：当前环境快照采集
- `precheck.py`：版本、截图引擎、执行模式、插件模式、目标窗口尺寸与 DPI 的预检骨架
- `package_manager.py`：安装、读取、预检、用户覆盖配置读写
- `server_config.py`：更新服务器上的共享平台包文件地址规划

后续可继续细化：

- 发布打包器
- 服务器元数据接口
- UI 共享平台页
- 运行时包级配置接管
- 运行前动态锚点快检
